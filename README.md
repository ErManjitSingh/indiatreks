# India Holiday Destinations — Treks

Premium trekking platform foundation for [treks.indiaholidaydestinations.com](https://treks.indiaholidaydestinations.com).

## Stack

- Next.js 15 (App Router) + React 19 + TypeScript
- Tailwind CSS v4 + shadcn-style UI primitives
- Framer Motion + GSAP + Lenis
- TanStack Query + Zustand + React Hook Form + Zod

## Commands

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
npm run format
```
